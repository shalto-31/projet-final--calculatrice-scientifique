# projet-final--calculatrice-scientifique

Ce projet est une calculatrice scientific rudimentaire fait en C++. il consiste de deux mode: un mode basic aisi qu'un mode avancé.

---

## les modes

| **Mode Basique**    | **Mode Avancé 2**                      |
|:-------------------:|:--------------------------------------:|
| Addition            | Puissance                              |
| Soustraction        | Racine Carrée                          |
| Multiplication      | Modulo (reste de division euclidienne) |
| Division            | factorielle                            |

---

## Usage

clonner le répo:

``` bash
git clone https://github.com/shalto-31/projet-final--calculatrice-scientifique.git
```

accéder au fishier:

~~~ bash
cd User/path/to/projet-final--calculatrice-scientifique
~~~

compiler:
~~~bash
g++ *.cpp -o build/calculatrice -Wall -Wextra
~~~

Run:

~~~
./build/calculatrice
~~~
